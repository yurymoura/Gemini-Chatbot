## üìã Description

**Gemini Chatbot** is a front-end study project focused on building an interactive, AI-powered conversational interface using **vanilla JavaScript**, modern CSS, and external APIs.  
The project simulates the core behavior of a real-world chat application while reinforcing best practices in **state management**, **asynchronous data handling**, and **event-driven UI architecture**.

The application was structured with clear separation between layout, styling, and logic, ensuring readability, scalability, and maintainability.

---

## üìò Technical Overview

The project emphasizes hands-on practice with **dynamic DOM manipulation**, **application state control**, and **user interaction feedback** without relying on front-end frameworks.

JavaScript is responsible for:
- Managing user input and form submission  
- Rendering chat messages dynamically  
- Simulating typing animations for AI responses  
- Controlling loading and error states  
- Persisting chat history and theme preferences using **LocalStorage**  

The chatbot integrates with the **Google Gemini API** through asynchronous `fetch` requests, demonstrating real-world API consumption, JSON handling, and error management. Responses are processed incrementally to improve perceived responsiveness and user experience.

The styling layer uses **CSS custom properties (variables)** to implement light and dark themes, with a modular CSS structure that promotes consistency and scalability across different screen sizes.

Overall, this project serves as a practical study of:
- Vanilla JavaScript UI architecture  
- Event-driven programming and DOM manipulation  
- Asynchronous API communication  
- Application state management without frameworks  
- Theme switching and persistent user preferences
